## Run Locally

Clonar el proyecto

```bash
  git clone https://github.com/alxmcr/bees-app
```

Ir donde se encuentra el directorio del proyecto.

```bash
  cd bees-app
```

Install dependencies

```bash
  npm install
```

Iniciar el Frontend server

```bash
  npm run start
```

Extra: Iniciar Backend server (Backend API)

```bash
cd frontend-test
npm install
npm run start
```
